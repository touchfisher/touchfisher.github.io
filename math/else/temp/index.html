# 题目

此题据传来自贴吧，证明者构造了一个如下的方程：

x2+x+1=0(1)x^{2}+x+1=0 \tag{1}x2+x+1=0(1)

显然 0 不是方程的根，于是可以在两侧同除xxx，得到：

x+1+1x=0(2)x+1+\frac{1}{x}=0 \tag{2}x+1+x1​=0(2)

将 (1) 和 (2) 式两侧做差，有：

x2−1x=0(3)x^2-\frac{1}{x}=0 \tag{3}x2−x1​=0(3)

显然有：

x=1(4)x=1 \tag{4}x=1(4)

此时将x=1x=1x=1 回代 (1) 式，得到一个明显错误的结论：

12+1+1=0(5)1^{2}+1+1=0 \tag{5}12+1+1=0(5)

可是，问题出在哪呢？
